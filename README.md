# Vicks'Creative Room - Tienda de artesanías

Proyecto de e-commerce desarrollado como parte del programa de desarrollo web.  
Permite visualizar productos artesanales, dar “likes” y realizar pedidos personalizados mediante formularios conectados a una API.

---

## Demo en vivo

Proyecto desplegado -- https://vickscreativeroom.netlify.app

## Tecnologías utilizadas

- React
- React Router DOM
- JavaScript (ES6+)
- CSS (BEM + Flexbox + Grid)
- Fetch API
- LocalStorage
- Vite

---

## Funcionalidades principales

- Catálogo de productos dinámico desde API
- Sistema de “likes” persistente con localStorage
- Paginación con “Mostrar más”
- Preloader durante carga de datos
- Manejo de errores con fallback local
- Formulario de pedidos personalizados
- Formulario de contacto
- Navegación entre páginas con React Router
- Diseño responsive (mobile-first)

---

## API utilizada

Se utilizó la API pública:
https://dummyjson.com/products
Para obtener productos.

---

## Aprendizajes

- Consumo de APIs con Fetch
- Manejo de estado con React Hooks
- Persistencia con localStorage
- Separación de lógica API/UI
- Componentización en React
- Manejo de formularios controlados
- Manejo de errores y UX con preloader
